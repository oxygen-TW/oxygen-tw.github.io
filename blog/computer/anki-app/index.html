學習利器介紹 - Anki

2022-08-02
computer notes
約 1666 字 預計閱讀 4 分鐘

知道 Anki 這款軟體已經有一段時間，但他操作起來仍然需要一點時間摸索，因為國考的關係被迫學習如何使用他XD 也順利地利用 Anki 的輔助考過國考了！ 便來介紹一下 Anki 如何使用，希望讓想要入門的人可以更快上手。

簡介

Anki 是一款輔助記憶方面學習的軟體(例如英文單字、解剖學等)，類似我們以前會使用的單字卡，在正面寫上中文，背面寫上英文這樣，他同時導入了間隔重複學習法(Spaced memory repetition)，主要是利用不斷的重複回憶增加記憶的強度的原理幫你分配學習的進度。

Anki 的軟體中答完題目可以依照對題目的熟悉度選擇重來、困難、良好、簡單這四種選擇，Spaced memory repetition的排程器會自動幫你決定同樣的題目應該要多久後再次出現，有興趣可以去研究他更詳細的排程原理與參考，但只適用於學習的話就不用管他，依照感覺選擇即可，依照我的經驗最常用的就是重來和良好這兩項而已。

下載

前往 https://apps.ankiweb.net/ 下載 Anki 主程式，主流作業系統都有支援。

  • Android：https://play.google.com/store/apps/details?id=com.ichi2.anki
  • iOS: https://apps.apple.com/tw/app/ankimobile-flashcards/id373493387
  • AnkiWeb：https://ankiweb.net/about
  • Source(GitHub)：https://github.com/ankitects/anki

*iOS因為開發平台每年都要付給Apple費用，所以 Anki for iOS 要用約台幣830元購買，如果不想花錢，可以到 AnkiWeb，功能沒有App那麼完善，但基本的複習功能都有了。

安裝好後進去會進入選擇設定檔的介面，可以依照自己要學習的不同東西建立不同的設定檔

新增牌組

點擊新增牌組就可以直接新增一個牌組，但如果你想要設定子牌組的話可以利用「::」做分割，例如：

新增 Deck::Subdeck，那麼牌組結構就會像

  • Deck
    • Subdeck

新增題目

點擊畫面上方的新增即可新增題目，Anki 預設主要有四種題目類型：

  1. 基本型
  2. 克漏字
  3. 反向卡片
  4. 輸入答案

首先從基本型開始介紹，他就是正面寫上問題，背面寫上答案，和我們用的紙本單字卡一樣。

使用起來類似這樣，正面

顯示背面答案後


克漏字可以在正面的題目敘述中挖空，而背面額外的欄位可以不填，那只是用來做額外的筆記用的

克漏字題目預覽


至於剩下的兩種，反向卡片其實就是基本型的反向，基本型是正面作為題目，但在反向基本型中背面也會被作為題目，例如；
  • CD23 <-> FcεRII
  • FcεRII <-> CD23

你只需要新增一張卡片，他就會自己幫你生成另一張反向的卡片

而輸入答案在背單字上很好用，他會自動檢查你輸入的單字和正確答案的差別


簡單的整理

類型 正面 背面 特點
基本型 文字題目 答案 普通的卡片
克漏字 克漏字挖空 不需要 可以在文字中挖空
反向型 答案/題目 答案/題目 一次生成正反兩組卡片
輸入答案 文字題目+輸入框 答案 檢查輸入與正確答案的差別

練習

複習時可以點選顯示答案翻開背面，或是鍵盤空白鍵也可以翻開背面 下面的數字代表新題目數量 + 正在學習中的數量 + 到期數量

新加入的題目會歸類到新題目，一但有被複習過，就會被加入學習中，而到期是指排程器覺得你應該再複習一次了，於是把題目放到到期的分類中，一但整個牌組都複習完，三個數字都應該是0。

顯示答案後下面會出現四個狀態，依據自己對題目的熟悉度選擇，基本上答錯我就會選重來，而上面的時間是指這個題目下一次出現的時間，你覺得題目越難，他就會出現的越頻繁直到學會，他的快捷鍵分別是重來-1, 困難-2, 良好-3, 簡單-4

擱置和暫停的區別

如果你想暫時跳過題目，可以點其他->擱置題目，那麼這個題目就會到第二天才出現。
如果你不想再做某一個題目，點其他->暫停題目，這個題目在你解除暫停前都不會出現。

同步資料

如果有到AnkiWeb申請免費的帳號，那就可以把牌組上傳到他們的伺服器，在不同台電腦、手機間同步複習狀態，這功能相當好用，出門在外用手機複習，回家再用電腦同步回來即可！


以上就是Anki的基本操作介紹，這套軟體本身挺複雜的，一開始入門真的看不太懂，加上他還有許多的擴充功能可以下載，有時間的話再寫文章介紹，熟悉這軟體最好的方法就是去玩玩看，玩久了自然就會理出頭緒！
